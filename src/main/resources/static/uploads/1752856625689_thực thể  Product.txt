- soldQuantity: int
- status: string

- description: string


thực thể : Brand

thực thể : User
thuộc tính : - userId: int
- username: string
- password: string

- role: string

thực thể : Cart
thuộc tính :  cartId: int

thực thể : Wishlist
thuộc tính :- wishlistId: int


thực thể : Compare
thuộc tính :- compareId: int

thực thể :  Order

MỐI QUAN HỆ :
| **Mối quan hệ**       | **Chi tiết**                                                                              |
| --------------------- | ----------------------------------------------------------------------------------------- |
| Product – Category    | Một Category có nhiều Product. Product thuộc về một Category. *(1 Category : N Product)*  |
| Product – Brand       | Một Brand có nhiều Product. Product thuộc về một Brand. *(1 Brand : N Product)*           |
| User – Wishlist       | Một User có một Wishlist. Wishlist chứa nhiều Product *(1 User : 1 Wishlist – N Product)* |
| User – Compare        | Một User có một Compare. Compare chứa nhiều Product *(1 User : 1 Compare – N Product)*    |
| User – Cart           | Một User có một Cart. Cart chứa nhiều Product *(1 User : 1 Cart – N Product)*             |
| User – Order          | Một User có nhiều Order. *(1 User : N Order)*                                             |
| Order – OrderDetail   | Một Order có nhiều OrderDetail. *(1 Order : N OrderDetail)*                               |
| OrderDetail – Product | Một OrderDetail gắn với 1 Product. *(1 Product : N OrderDetail)*                          |
| User – Rating         | Một User có thể đánh giá nhiều Product *(1 User : N Rating)*                              |
| Product – Rating      | Một Product có thể có nhiều Rating *(1 Product : N Rating)*                               |
| User – Message        | Một User có thể gửi/nhận nhiều Message *(1 User : N Message gửi + N Message nhận)*        |
| Shipping – Order      | Một Order gắn với một Shipping (khu vực vận chuyển) *(1 Shipping : N Order)*              |
